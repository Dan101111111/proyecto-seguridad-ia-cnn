# Data Directory

Esta carpeta contiene los datasets para entrenamiento y validación del modelo CNN.

## Estructura recomendada:

```
data/
├── raw/           # Datos sin procesar
├── processed/     # Datos preprocesados
├── train/         # Conjunto de entrenamiento
├── validation/    # Conjunto de validación
└── test/          # Conjunto de prueba
```

## Notas:
- Los archivos de datos grandes no se suben al repositorio (ver .gitignore)
- Agregar aquí tus datasets de imágenes para entrenamiento
